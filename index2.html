Ans: there are several ways to select an element in the DOM. Such as: Using getElementById,getElementsByClassName,getElementsByTagName,querySelector and querySelectorAll.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

ans:innerHTML, innerText, and textContent are properties of the DOM.But there is a difference among them.

  • innerHTML: Gets or sets HTML content (tags included).
  • innerText: Gets or sets visible text content, accounting for CSS.
  • textContent: Gets or sets raw text content, without HTML tags or CSS effects.

Question-3: What is event delegation in the DOM?

Event delegation is a technique in JavaScript where we delegate the responsibility of handling an event to a parent element. By doing so, we avoid attaching multiple event listeners to individual elements, especially when dealing with a large number of similar elements, such as a list or a table.

Question-4: What is event bubbling in the DOM?

Event Bubbling is a concept in the DOM. It happens when an element receives an event, and that event bubbles up to its parent and ancestor elements in the DOM tree until it gets to the root element

Question-5: How do you create, add, and remove elements using JavaScript?

JavaScript provides several methods for dynamically manipulating the Document Object Model (DOM) by creating, adding, and removing elements. Here's a breakdown of the core techniques:1. Creating Elements:document.createElement(tagName):This is the fundamental method. It creates a new HTML element specified by the tagName . 2. Adding Elements:parentNode.appendChild(childNode):Appends a childNode as the last child of parentNode.Example: document.body.appendChild(newDiv); parentNode.insertBefore(newNode, referenceNode):Inserts a newNode before a referenceNode within a parentNode.parentNode.append(...nodes): This is a more modern method that allows you to append several nodes or strings at the end the parent node.parentNode.prepend(...nodes):This method allows you to insert nodes or strings at the begining of the parent node. 3. Removing Elements:parentNode.removeChild(childNode):Removes a childNode from its parentNode.childNode.remove(): Removes the childNode from the DOM directly.
